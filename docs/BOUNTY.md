# PlantGuide bounties (MergeOS MRG)

PlantGuide pays contributors in **MRG** through the MergeOS bounty program.

## Before you start

1. Star https://github.com/mergeos-bounties/PlantGuide  
2. Star https://github.com/mergeos-bounties/mergeos  
3. Comment on the PlantGuide issue: `I claim this bounty`  
4. Comment on MergeOS [Claim Token #1](https://github.com/mergeos-bounties/mergeos/issues/1) with the issue link  
5. Open a PR to **PlantGuide** `master` (`Fixes #N`)

### Full guides (all languages)

See **[docs/guides/README.md](guides/README.md)** for EN / VI / 中文 / 日本語 / 한국어 / ES / FR / DE / PT / ID / TH / HI / AR / RU.

## Payout scale

| Scope | Typical MRG |
| --- | ---: |
| Small / good-first | 25 |
| Medium feature | 50 |
| Large system piece | 100 |
| Extra-large | 200 |

Issue titles may show marketing amounts; **admin ledger pays 25/50/100/200** after review.

## Domains we fund

- Species catalogs & care cards  
- Vision models for leaf/plant photo ID  
- Disease / pest detection  
- Watering & fertilizing schedules  
- FastAPI / mobile SDK for plant apps  
- Multilingual care tips (VI/EN)  
- Evaluation datasets & metrics  

## Evidence

- Tests + command output for backend/ML  
- Screenshots for UI demos  
- Dataset license notes for any new media
