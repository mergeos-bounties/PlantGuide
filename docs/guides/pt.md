## Passos

1. Star PlantGuide e mergeos.
2. Issue species-pack: https://github.com/mergeos-bounties/PlantGuide/issues?q=is%3Aissue+is%3Aopen+label%3Aspecies-pack
3. Comente: `I claim this bounty`
4. Também MergeOS #1.
5. ≥2 fotos originais.
6. PR species + sample.
7. `Fixes #<issue>` + fotos.
8. Testes locais.
